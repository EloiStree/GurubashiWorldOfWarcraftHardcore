You have install a offline server at home or want to connect to a 3.3.5a Wotlk server ?

-----------

- Download a Torrent software: https://www.bittorrent.com
- Find a torrent hosts link: https://www.warmane.com/download
- Move it to a folder like `C:\Gurubashi\`
- Got in `C:\Gurubashi\World of Warcraft 3.3.5a\Data\enUS`
- Change the file `realmlist.wtf` to `set realmlist 192.168.1.88`
  - Or the server name / public ip
- Launch the exe and enjoy
